## Инструкция по запуску приложения

### Linux

```bash
cd dash_app_baikalsk
bash setup.sh
python3 app.py
```

После запуска приложение будет доступно в браузере:
**[http://127.0.0.1:8050](http://127.0.0.1:8050)**


## Кратко об архитектуре

Приложение реализовано на Python с использованием Dash и Plotly и работает как локальный веб-сервер.

* **app.py** - точка входа: инициализация приложения, базовый шаблон и запуск сервера
* **layout.py** - структура интерфейса (шапка, вкладки, страницы)
* **callbacks.py** - обработка взаимодействия пользователя (переключения, фильтры)
* **data.py** - загрузка и подготовка данных (CSV, GeoJSON)
* **parsing.ipynb** - сбор данных и их обработка
* **gee_ndvi.js** - рассчет NDVI
* **tiff_to_csv.py** - конвертация GeoTIFF в CSV
* **analysis.ipynb** - рассчет аналитики
* **pages/** - отдельные разделы (транспорт, здания, функционал, NDVI)
* **assets/** - стили
* **data/** - исходные данные
* **analysis_results/** - предрасчитанная аналитика
